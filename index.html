Plasma Density Profile Plotter

Build a plasma-density profile with fully editable ramps, plateau, styling, annotations, and export-friendly linear and logarithmic views for GitHub Pages.

JupyterLab-inspired light theme

Profile controls

Set the geometry, ramp behavior, density unit, and plasma profile parameters. The horizontal axis uses μm throughout.

Right side

Control the decay from the plateau back into vacuum.

Enable right ramp

Plots

Choose whether the image buttons copy to the clipboard or export to local files. Both PNG and SVG follow the selected action mode, and you can also export the plotted data as a NumPy .npz package.
